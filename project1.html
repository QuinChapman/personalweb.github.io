Project Background


I took my first web design class in Spring 2019 where I was taught the basics of HTML, CSS, and a bit of JavaScript.


Based on that knowledge, I decided to make an attempt at creating a random password generator, a web application that generates passwords and stores them locally within the browser.


In my first iteration of the random password generator, I held little to no knowledge of basic graphic design principles and essential usability heuristics, and it showed!


Having taken classes these past two semester that emphasized the importance of such principles and heuristics, I was able to apply the knowledge I acquired and create a user-focused and responsive website!
